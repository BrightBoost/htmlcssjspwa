Use the steps from the demo to turn this page into a PWA

  1. Ensure your site is mobile-optimized; test with Chrome's Lighthouse tool.
  2. Register a service worker: In your main JS, check for service worker support and register serviceworker.js.
  3. Add Workbox to your project: In serviceworker.js, import Workbox from a CDN using `importScripts`.
  4. Implement caching: Use Workbox's routing module to register routes and define caching strategies.
  5. Add the icons with: npx pwa-asset-generator icon.png icons
  6. Create a manifest.json file: Include app details, icons, and ensure it's linked in your HTML.
  7. Test your PWA: Re-run Lighthouse in Chrome DevTools to ensure PWA criteria are met.
